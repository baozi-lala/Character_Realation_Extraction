What is Visual Entailment?

Visual Entailment(VE) is a multi-modal inference task focusing on recognizing if image premise implies sentence hypothesis.
check_circle

Entailment

Entailment holds if there is enough evidence in P to conclude that H is true.

help

Neutral

The relation is neutral if the evidence in P is insufficient to draw a conclusion from H.

cancel

Contradiction

Contradiction is concluded wherever H contradicts P.
